NOTE: readme.txt
------------------------------------------------------------------------------------------------------

Codes for estimating 4-parameter time-varying intervention effects model described in
Xu et al (2017) Semiparametric estimation of time-varying intervention effects using recurrent event data.
Statistics in Medicine 2017;36(17):2682-2696

------------------------------------------------------------------------------------------------------

"Functions4ParG1": program includes functions needed in the model estimation.

"FourParModelG1": codes for estimation of an extended AG model with time-varying intervention effect 
captured by a 4-parameter monotonic function.

"Example_data_2023.csv": a (hypothetical) recurrent events dataset for demonstration of data format and for testing only.
